**Spacing scale (DS-01) — add after the Shadow section:**
```css
  /* Spacing Scale — consistent layout rhythm */
  --spacing-page-x: 1rem;        /* Horizontal page padding (mobile) */
  --spacing-page-x-lg: 1.5rem;   /* Horizontal page padding (desktop) */
  --spacing-section-gap: 1.25rem; /* Gap between sections/widget groups */
  --spacing-card-padding: 1rem;   /* Internal card padding */
  --spacing-card-gap: 0.75rem;    /* Gap between cards in a grid */
```

**Typography scale documentation (DS-03) — add as a CSS comment after the Fonts section:**
```css
  /* Typography Scale (reference — use Tailwind utility classes)
   * --------------------------------------------------
   * Page title:    text-xl  font-semibold (20px/28px)  — PageHeader h1
   * Section title: text-lg  font-semibold (18px/28px)  — card group headings
   * Card title:    text-base font-medium  (16px/24px)  — WidgetCard, meal card headings
   * Body:          text-sm               (14px/20px)  — general text
   * Caption:       text-xs               (12px/16px)  — labels, timestamps, subtitles
   * --------------------------------------------------
   * All headings: normal case (no text-transform: uppercase)
   * Weight: semibold (600) for titles, medium (500) for card titles, normal for body
   */
```

This is a documented reference scale — not CSS custom properties — because Tailwind's built-in `text-*` and `font-*` utilities already handle typography. The comment serves as Phase 13's guide for consistent sizing.
