  <action>
    Add smart default weight pre-fill to the check-in page:

    1. Import `useQuery` (already imported from "convex/react") and add a query for the latest check-in:
       ```typescript
       const latestCheckIn = useQuery(api.checkIns.getLatestCheckIn);
       ```
       Note: `getLatestCheckIn` is an existing query in `convex/checkIns.ts` that returns the user's most recent check-in with all fields including `weight`.

    2. Add a `useEffect` that sets the weight default when the query loads:
       ```typescript
       useEffect(() => {
         if (latestCheckIn?.weight && !methods.getValues("weight")) {
           methods.setValue("weight", latestCheckIn.weight);
         }
       }, [latestCheckIn, methods]);
       ```
       The guard `!methods.getValues("weight")` prevents overwriting if the user has already typed a weight value.

    3. Import `useEffect` from React (add to existing import if not already there).

    4. Do NOT change the `defaultValues` in `useForm` -- the weight default is still `undefined`. The `useEffect` approach handles the async loading correctly since Convex queries return `undefined` while loading.

    5. Do NOT add any visual indicator like "pre-filled from last check-in" -- just silently set the value. The user sees the field already filled and can change it.
  </action>
